# Start

## How to start?

1. 노드 모듈 설치하기

```
$ yarn
```

2. 루트 디렉토리에 uploads 폴더 만들기

```
$ mkdir uploads
```

3. 서버 실행하기

```
$ node server
```

4. 에디터 실행하기

```
$ yarn start
```
